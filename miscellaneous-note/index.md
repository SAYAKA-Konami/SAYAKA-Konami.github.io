# Miscellaneous Note


# 杂记

## 想象力

我简单的收拾了一下行李，便想到宿友的台式机前玩玩游戏。开机键还未触及前就看到了放在桌上的草稿纸。上面略微整齐的摆着几道算术式和零散的一些词诸如“厕所”、“空调”......我拿起来问我那位考公的宿友，说这行测的草稿会不会有些正式，他说买了台式机的那位宿友要去开麻将馆。我笑出了声，毕竟我没在上面看到租金二字。他说他一年能存大概一万，那位考公的宿友补充道。我再次笑出了声。他说他好像存了二十万，宿友一句话打断了我的笑意，除了和数学运算得出的结果不符之外，还顺带把我倒回了不久前相遇的时候。

彼时我开玩笑地嫌弃着他几乎快摔碎的手机，并建议他换一个。他倒没接话，只是顺嘴问了句我新买的手机用起来如何。我认真的说了一下优缺点，扭头一看他已然在浏览14pro。我正要想着拼出几句无厘头的话语用于激将，他便付了钱。

我恼怒的把问题抛到了应试教育的头上。一来是因其害得我丢了些许浪漫——我是指想象力。二来是其害得我知道了这些事情——想象一下一群戴着学士帽的纤夫，你便也会笑出声来。
